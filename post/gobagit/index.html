gobagit - A tool to work with the BagIt format

on Sat, Jul 06, 2019

gobagit is a command line tool and golang library to work with the BagIt format I am writing currently.

I really like the bagit format because it is simple, a standard (RFC 8493) and widely used. gobagit is available in version 0.2.0 and under heavy development, check the dev branch.

I’d like to get some feedback, so drop me an issue or star on github.

https://github.com/steffenfritz/bagit

You don’t have to build the tool yourself, just download a pre-built from the release page

https://github.com/steffenfritz/bagit/releases
