siegfried - a signature-based file format identification tool

I wrote a PKGBUILD file for siegfried, the “signature-based file format identification tool, implementing the National Archives UK’s PRONOM file format signatures and freedesktop.org’s MIME-info file format signatures

Install it on Arch Linux with

yaourt -S siegfried

The package is called siegfried, the binary sf. After the installation run sf -update to fetch a recent version of the signature database.

Siegfried is developed by Richard Lehane. For more info check http://www.itforarchivists.com/siegfried

Comments on AUR or via email are highly appreciated!
