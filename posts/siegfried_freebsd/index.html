Siegfried on FreeBSD

Last weekend I ported siegfried to FreeBSD. Siegfried is an advanced file identification tool, using the National Archives UK’s PRONOM and freedesktop’s MIME-info file format signatures. It’s written in Go and compared to other file id tools pretty fast.

The port can be found on github, it is waiting to be added to the ports tree.
